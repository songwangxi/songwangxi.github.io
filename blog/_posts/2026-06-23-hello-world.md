## 为什么选择 GitHub Pages？

- **完全免费** —— 无需任何费用
- **支持自定义域名** —— 可以用自己的域名
- **Markdown 写作** —— 专注内容，无需关心排版
- **版本控制** —— 每次修改都有记录
- **全球加速** —— GitHub 的 CDN 让访问更快
